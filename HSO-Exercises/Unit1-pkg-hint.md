Unit 1 - pkg [HINT]
---

`pkg` is FreeBSD's package management suite.

### **Exercises**

  * Search for the package `figlet`
  * Install the package `figlet`
  * Run figlet and have it print out "hello"
  * Remove the package figlet

### Hints

    ~ pkg search figlet
    ...

    ~ sudo pkg install figlet
    ...

    ~ figlet "hello!"
